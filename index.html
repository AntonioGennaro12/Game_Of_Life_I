Ayuda para el Juego de la Vida de Conway

1. Elige la cantidad de Filas y Columnas del Tablero.

2. Selecciona la cantidad de Civilizaciones que quieres mostrar (1 a 6).

.. Si eliges una (1) civilización, entonces puedes elegir el Algoritmo.

... Hay 7 algoritmos pre-definidos y una opción a Definir por el Usuario.

... Al seleccionar "User Defined" y confirmar, se despliega otro menú:

.... - 1er valor: Mínimo de Vecinos para permanecer Vivo

.... - 2do valor: Máximo de Vecinos para permanecer Vivo

.... - 3er valor: Número de Vecinos para renacer

.... - 4to valor: Opcional. 2do Número de Vecinos para renacer,

...... si no usas el 2do valor, cargar el mismo valor que el 3ro

... Por último está la opción de "Tablero Vacío" en la que se pueden ubicar

.... células vivas y ver como evolucionanan con el último algoritmo elegido.

.... Se detiene el juego, haciendo click sobre la celda se activan las que

..... se desean y luego se renauda para ver su evolución.

.. Si es más de una civilización (2 a 6), entonces se aplica un algorimo fijo

... en forma progresiva: 12/2, 23/3, 34/4, 45/5, 56/6 y 67/7, según el número.

3. Una vez configurado, confirmar y aparecerá "Play Game"

4. Con el botón "Play Game" se inicia y detiene el "Juego"

5. Cuando se lo detiene, puede ser reiniciado con la misma configuración

.. o bién re-armar el tablero y cambiar la configuración...

... Fin. (click en "x" para cerrar). ©AG2023
