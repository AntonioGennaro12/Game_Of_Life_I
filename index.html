Ayuda para el Juego de la Vida de Conway

1. Elige la cantidad de Filas y Columnas del Tablero.

2. Selecciona la cantidad de Civilizaciones que quieres mostrar.

.. si es una sola civilización (1) puedes elegir el Algoritmo a aplicar

... hay 7 algoritmos pre-definidos y uno Definido por el Usuario

... cada algoritmo define 3 valores y opcionalmente un cuarto valor.

... al seleccionar "User Defined" y confirmar se despliega otro menú

.... 1er valor: Mínimo de Vecinos para permanecer Vivo

.... 2do valor: Máximo de Vecinos para permanecer Vivo

.... 3er valor: Número de Vecinos para re-Nacer

.... 4to valor: Opcional. 2do Número de Vecinos para re-Nacer

..... si no usas el 2do valor, cargar el mismo valor que el 3ro

.. si es más de una civilización (2 a 6) entonces el algorimo es fijo

.. aplicando la configuración 12/2, 23/3, 34/4, 45/5, 56/6 y 67/7

3. Una vez configurado, confirmar y aparecerá "Play Game"

4. Con el botón Play Game se inicia y detiene el "Juego"

5. Cuando se lo detiene, puede ser reiniciado con la misma configuración

.. o bién re-armar el tablero y cambiar la configuración...

fin. (click en "x" para cerrar).
