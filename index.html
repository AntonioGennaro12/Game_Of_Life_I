Ayuda para el Juego de la Vida de Conway

1. Elige la cantidad de Filas y Columnas del Tablero, cuantas más mejor.

2. Selecciona la cantidad de Colonias que quieres mostrar (1 a 7) o sino un

.. Tablero "vacío" dónde ubicar celdas vivas y ver como evolucionan.

.. Si eliges una (1) colonia o "Tab. Vacío", podrás elegir el Algoritmo.

... Hay 7 algoritmos pre-definidos y una opción a Definir por el Usuario.

... Al seleccionar "User Defined" y confirmar, se despliega otro menú:

.... - 1er valor: Mínimo número de Vecinos para permanecer Vivo

.... - 2do valor: Máximo número de Vecinos para permanecer Vivo

.... - 3er valor: Número de Vecinos para renacer

.... - 4to valor: Opcional. 2do Número de Vecinos para renacer,

...... si no usas el 2do valor, cargar el mismo valor que el 3ro

... En la opción "Tablero Vacío" se hace click sobre la celda para activar

.... o desactivar y se renauda para ver su evolución. Mientras están en /p>

..... movimiento se puede clickear para agregar o quitar y ver qué sucede.

.. Si es más de una civilización (2 a 7), se aplica un algorimo fijo en forma

... progresiva: 23/3, 23/36, 12/2, 12/24, ...34/46, según el número de colonias.

3. Una vez configurado, confirmar y aparecerá "Play Game"

4. Con el botón "Play Game" se inicia y detiene el "Juego"

5. Cuando se lo detiene, puede ser reiniciado desde dónde quedó con la misma

.. configuración, o bién re-armar el tablero y cambiar la configuración...

... Fin. (click en "x" para cerrar). ©AG2023
